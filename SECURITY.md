# Security Policy

## Supported Versions

| Version      | Supported |
|--------------|-----------|
| 2.0.<latest> | ✅         |
| 1.4.<latest> | ✅         |
| older        | ⛔️        |

## Reporting a Vulnerability

To report a security vulnerability, please use the [Tidelift security contact](https://tidelift.com/security).
Tidelift will coordinate the fix and disclosure.
